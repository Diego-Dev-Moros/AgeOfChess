# ♟️ Age of Chess — Dice-Based Strategy Game

A modern reimagining of chess that blends classic strategy with dice-based combat, RPG mechanics, and pixel art aesthetics.

## 🎮 Concept

Age of Chess preserves the core rules of traditional chess while introducing:

- 🎲 Dice-based combat system (inspired by Risk)
- ❤️ Health and damage mechanics for each piece
- 🃏 Buff and debuff cards that alter gameplay
- 🌍 Terrain modifiers affecting the entire board
- 👑 A unique "Divinity System" for the King

The result is a deeper, more dynamic, and unpredictable strategy experience.

## 🧠 Core Features

- Turn-based tactical gameplay
- Probabilistic combat resolution
- Modular card system (Buffs / Debuffs / Terrain)
- Scalable game architecture (designed for extensibility)
- Pixel art visual identity with a cohesive art direction

## 🎨 Visual Style

- Pixel art (SNES-inspired with modern polish)
- Medieval fantasy aesthetic
- Strong emphasis on readability and gameplay clarity
- Unified art direction across cards, pieces, and board

## ⚙️ Tech Stack

- 🎮 Engine: Godot (2D)
- 🎨 Assets: AI-assisted + custom pixel art pipeline
- 🧠 Design system documented in Obsidian
- 📦 Modular architecture for gameplay systems

## 📂 Project Structure

- `game/` — Core gameplay logic
- `assets/` — Sprites, cards, tiles, UI
- `docs/` — Game design & art direction
- `prompts/` — AI generation prompts
- `systems/` — Combat, cards, terrain logic

## 🚧 Status

Early development — core systems and visual direction are being defined.

## 🔮 Vision

To create a unique tactical experience where:
- Strategy meets randomness
- Every match feels different
- Small pieces can defeat giants
- The King becomes a true gameplay centerpiece
