# TURN SYSTEM

---

# 1. TURN STRUCTURE

Each turn:

1. Select piece
2. Move
3. Combat (if triggered)
4. Apply modifiers
5. Evaluate board

---

# 2. PLAYER ACTIONS

A player can:

- Move one piece
- Attack via movement
- Use King ability (movement-based)

---

# 3. VALID MOVES

- Must follow chess rules
- Cannot leave King in check

---

# 4. PRIORITY

Actions resolve in order:

1. Movement
2. Combat
3. Effects

---

# 5. CHECK VALIDATION

After each move:

- Check state must be evaluated

---
