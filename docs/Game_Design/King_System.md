# KING SYSTEM — DIVINITY

---

# 1. OVERVIEW

The King is a special unit:

- Does not use combat system
- Cannot take damage
- Controls tempo via Divinity

---

# 2. DIVINE ATTACK

If Divinity > 0:

- Target dies instantly
- Divinity -1

---

# 3. COMMITMENT

After attacking:

→ King cannot move next turn

---

# 4. DIVINITY STATES

| Value | Effect |
|------|--------|
| 3–1 | Instant kill |
| 0 | Cannot attack |
| -1 to -3 | Cowardice |
| ≤ -3 | Allied HP = 1 |

---

# 5. RECOVERY

If no attack for 3 turns:

→ +1 Divinity

---

# 6. OVERUSE

If attacking twice in 2 turns:

→ -2 Divinity

---

# 7. DESIGN ROLE

The King is:

- A tempo weapon
- A risk amplifier
- A late-game decision tool
