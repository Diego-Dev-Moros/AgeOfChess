# Visual Identity

---

# 1. Core Visual Themes

Age of Chess combines four visual themes:

1. Chess as ritual combat.
2. Stone and metal as the base material language.
3. Dice as controlled fate.
4. Terrain as a magical battlefield condition.

The identity should be recognizable even without text:

- Carved chess silhouettes.
- Arcane dice motifs.
- Gold and parchment card structure.
- Strong faction glow.
- Tactical board clarity.

---

# 2. Shape Language

| System | Shape Language |
| --- | --- |
| Light faction | Upright, clean, noble, angular but not sharp |
| Dark faction | Heavier, chipped, hooked, shadowed |
| Buffs | Open curves, upward motion, rings, vines, radiant particles |
| Debuffs | Broken curves, downward motion, cracks, thorns, smoke |
| Terrain | Broad environmental shapes, not character-like symbols |
| Dice | Cubic, beveled, rune-marked, high contrast pips |

---

# 3. Symbol Consistency

Recurring symbols:

- Six-sided dice.
- Chess silhouettes.
- Runes.
- Cracks in stone.
- Gold trim.
- Magical particles.

Symbol rules:

- Dice symbols must be subtle in backgrounds unless dice are the focal asset.
- Runes must use consistent blocky pixel glyphs.
- Chess silhouettes must stay readable before decorative damage or glow.
- Do not invent a new symbol family for each card.

---

# 4. Recognition Principles

Every asset must communicate in this order:

1. Asset category.
2. Gameplay function.
3. Faction or card type.
4. Material and atmosphere.
5. Decorative details.

For example, a buffed dark knight should first read as a knight, then as dark faction, then as buffed, then as basalt or metal.
