# Pieces Asset Design Index

---

# 1. Purpose

This folder contains one visual specification per chess piece.

All piece sprites use:

- 48x48 px canvas.
- 2 px transparent safety margin.
- 1 px dark outer outline.
- Top-left lighting.
- Light and dark faction variants.
- Separate overlays for damaged, buffed, debuffed, selected, legal move, and attack target states.
