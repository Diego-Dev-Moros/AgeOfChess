# Age of Chess Art Assets

This folder contains the repo-level art layout described in the Obsidian asset pipeline.

Godot loads the mirrored runtime copy from `game/assets/art` because the active Godot project root is `game/project.godot`.

## Runtime Groups

- `board/`: terrain board textures at 512x512.
- `cards/buffs/`: buff card previews at 192x288.
- `cards/debuffs/`: debuff card previews at 192x288.
- `cards/terrain/`: terrain card previews at 192x288.
- `dice/`: 32x32 dice icons by kind and face.
- `pieces/white/` and `pieces/black/`: 48x48 piece sprites.
